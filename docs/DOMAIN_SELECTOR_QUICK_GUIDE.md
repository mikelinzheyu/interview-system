## 🔍 搜索功能详解

### 支持的搜索方式

**方式1：中文搜索**
```
输入：计算机    → 找到：计算机科学与技术
输入：金融      → 找到：金融学
输入：法律      → 找到：法学
```

**方式2：拼音搜索**
```
输入：jisuanji  → 找到：计算机科学与技术
输入：jinrong   → 找到：金融学
输入：falv      → 找到：法学
```

**方式3：拼音首字母**
```
输入：jsy       → 找到：计算机科学与技术
输入：jrx       → 找到：金融学
输入：fx        → 找到：法学
```

**方式4：英文搜索**
```
输入：computer   → 找到：计算机科学与技术
输入：finance    → 找到：金融学
输入：law        → 找到：法学
```

### 搜索建议机制

搜索时，系统会显示最相关的5条建议：
```
搜"计" →
  ✓ 计算机科学与技术
  ✓ 金融学
  ✓ 经济学
  ✓ 计算机与互联网
  ✓ 金融科技
```
