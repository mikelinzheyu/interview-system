## 🎨 整体设计理念

### 设计目标

| 维度 | 目标 | 说明 |
|------|------|------|
| **视觉** | 现代化、高端、专业 | 参考 test6 的渐变、人物卡片风格 |
| **功能** | 快速导航、信息密集 | 用户一眼看到所有功能入口 |
| **交互** | 流畅、有反馈 | 悬停动画、过渡效果 |
| **性能** | 加载快速 | <2s 首屏加载时间 |
| **转化** | 提升用户留存 | 让用户快速开始使用 |

### 设计原则

```
1. 优先级清晰      - 重要内容突出，次要内容淡化
2. 信息层级       - 欢迎 > 统计 > 功能 > 推荐 > 活动
3. 视觉对比       - 渐变背景 + 玻璃态卡片
4. 交互反馈       - 鼠标、触摸都有清晰反馈
5. 无障碍设计     - 颜色对比度、文字可读性
```

## ✨ 交互体验

### 悬停效果

```css
/* 卡片悬停升起 */
.feature-card:hover {
  transform: translateY(-6px);
  box-shadow: 0 12px 35px rgba(0, 0, 0, 0.25);
  background: rgba(255, 255, 255, 1);
}

/* 按钮悬停 */
.btn:hover {
  transform: scale(1.05);
  box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
}

/* 统计卡片悬停 */
.stats-card:hover {
  transform: scale(1.02);
}
```

### 过渡动画

```css
/* 页面加载入场 */
@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.section {
  animation: slideUp 0.6s ease-out;
  animation-delay: calc(var(--index) * 0.1s);
}

/* 加载动画 */
@keyframes shimmer {
  0% { background-position: -1000px 0; }
  100% { background-position: 1000px 0; }
}
```

### 点击反馈

```javascript
// 防抖点击
function handleCardClick(e) {
  e.preventDefault();

  // 涟漪效果
  const ripple = document.createElement('span');
  ripple.classList.add('ripple');
  e.currentTarget.appendChild(ripple);

  // 延迟导航
  setTimeout(() => {
    router.push(feature.route);
  }, 200);
}
```

## 📱 响应式设计

### 断点设置

```
Desktop:  ≥ 1200px  (4列布局)
Tablet:   768-1199px (2-3列布局)
Mobile:   < 768px   (1列布局)
```

### 各端口设计

#### 桌面 (1920px+)
```
Hero: 并排布局, 完整文案
Stats: 4列
Features: 5列 (首屏 3个, 下滑 2个)
Insights: 60/40 分割
```

#### 平板 (768-1199px)
```
Hero: 上下堆叠, 文案缩小
Stats: 2x2 网格
Features: 2-3列
Insights: 垂直堆叠
```

#### 手机 (< 768px)
```
Hero: 完全上下堆叠
Stats: 1列, 可滚动
Features: 1列
Insights: 单列 (图表缩小)
Activity: 全宽度
```
