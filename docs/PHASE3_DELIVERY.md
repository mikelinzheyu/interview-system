## 📞 支持信息

### 快速问题排查

**Q: 表情选择器不显示?**
- A: 检查 localStorage 是否可用
- A: 检查浏览器控制台错误
- A: 清除浏览器缓存重试

**Q: Ctrl+F 不工作?**
- A: 确保焦点在页面上
- A: 检查是否有其他快捷键冲突
- A: 尝试在 Chrome/Firefox 中测试

**Q: @mention 不出现?**
- A: 输入 @ 后等待 200ms
- A: 检查是否有空格
- A: 确保成员列表已加载
