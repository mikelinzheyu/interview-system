## 💡 核心特性

### 1. 自动性价比计算
```
实际可支配收入 = 平均薪资 - 生活成本
消费占比 = 生活成本 / 平均薪资 (越低越好)
```

**例如**:
- 北京: ¥28K薪资, ¥8K成本 → ¥20K实际 (28.6%消费)
- 杭州: ¥25K薪资, ¥5.5K成本 → ¥19.5K实际 (22%消费)

### 2. 多维度评分系统
```
薪资竞争力 = 该城市薪资 / 最高薪资 × 100
生活成本得分 = (1 - 成本占比×0.5) × 100
机会得分 = 该城市职位数 / 最多职位数 × 100
综合得分 = (竞争力×0.4 + 成本×0.3 + 机会×0.3) / 20
```

### 3. 智能推荐系统
```
评估维度:
- 薪资 > 全国平均 && 成本占比 < 35% → 💚 推荐
- 薪资 > 全国平均 && 成本占比 > 40% → 💙 较好
- 薪资 ≤ 平均 && 成本占比 < 35% → 💜 考虑
- 其他情况 → 💛 参考
```

## 🎯 使用场景

### 场景1: 我想了解该专业薪资水平
1. 完成兴趣评估
2. 选择感兴趣的推荐专业
3. 点击 **薪资分析** 标签
4. 查看**概览卡片**了解平均薪资、中位数等

### 场景2: 我想对比不同城市的发展机会
1. 在薪资分析标签页
2. 向下滑动到 **城市对比** 区域
3. 使用排序按钮对城市进行排序
4. 对比每个城市的薪资、成本、评分

### 场景3: 我想找性价比最高的城市
1. 在城市对比区域
2. 点击 **按性价比排序** 按钮
3. 第一个卡片就是实际收入最高的城市
4. 查看该城市的推荐提示

### 场景4: 我想了解职级进阶和薪资成长
1. 在薪资分析标签页查看 **经验成长** 部分
2. 了解每个职级的薪资水平
3. 查看 **成长曲线** 了解7年的薪资增长趋势
4. 结合 **学位对比** 规划学历投资回报

## 🐛 常见问题

**Q: 为什么看不到薪资数据？**
A: 需要先完成推荐评估并获得推荐结果，才能查看薪资分析。

**Q: 数据是实时的吗？**
A: 目前使用样本数据。将来可通过API接入实时数据源。

**Q: 能导出薪资数据吗？**
A: 当前版本不支持，可在Phase 4.4中增加导出功能。

**Q: 城市列表能自己添加吗？**
A: 需要修改 `salary.js` 中的 `sampleSalaryData` 对象。
