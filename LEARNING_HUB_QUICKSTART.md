## 🚀 如何使用

### 1. 开发环境启动
```bash
cd frontend
npm install  # 如果需要
npm run dev
```

### 2. 访问新UI
```
浏览器打开: http://localhost:5173/questions/hub
```

### 3. 测试流程

#### 新用户流程:
1. 进入首页 → 看到"为您推荐"
2. 选择推荐的学科 → 进入学科大类
3. 选择专业 → 看到专业卡片
4. 选择领域 → 进入详情页
5. 查看学习路径 → 点击"开启学习"

#### 快速搜索:
1. 按 Ctrl+K
2. 输入 "数据结构"
3. 看到搜索结果
4. 点击结果快速导航

#### 查看进度:
1. 点击顶部 "学习进度"
2. 查看统计和目标
3. 返回继续学习
