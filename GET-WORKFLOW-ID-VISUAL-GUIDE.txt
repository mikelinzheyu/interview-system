================================================================================
                   如何获取 Workflow API ID - 可视化指南
================================================================================

你现在有的 (错误的):
  560EB9DDSwOFc8As
  这个来自: https://udify.app/workflow/560EB9DDSwOFc8As

我们需要的 (正确的):
  [某个 UUID 或内部 ID]
  这个来自: https://cloud.dify.ai/app/...../workflows/[THIS]

================================================================================
步骤 1: 打开浏览器并登录 Dify
================================================================================

URL: https://cloud.dify.ai
点击: 登录 (如果还没登录)

================================================================================
步骤 2: 找到你的应用
================================================================================

在左边菜单中:
  - "我的应用" 或 "Applications"
  - 找到 "Workflow1" 或 "AI面试官-工作流1"

如果有很多应用，搜索 "Workflow" 或 "生成问题"

================================================================================
步骤 3: 打开 Workflow
================================================================================

点击应用名称 → 进入应用详情页面

看到工作流的编辑界面

================================================================================
步骤 4: 看地址栏 (重要!)
================================================================================

当你在编辑 Workflow1 时，浏览器地址栏应该显示:

  https://cloud.dify.ai/app/[SOME-ID]/workflows/[WORKFLOW-ID]
                                                  ^^^^^^^^^^^^^^^^^^
                                                  这就是我们需要的!

例如，可能看起来像:

  方案 A (UUID 格式):
    https://cloud.dify.ai/app/abc123def456/workflows/
      5a9c3d4b-7e9f-4f2a-8c1d-2e5f6a8b3c1d

  方案 B (数字格式):
    https://cloud.dify.ai/app/abc123def456/workflows/
      1234567890abcdef

  方案 C (其他格式):
    https://cloud.dify.ai/app/abc123def456/workflows/
      workflow-abc-123-xyz

================================================================================
步骤 5: 复制 Workflow ID
================================================================================

看 "workflows/" 之后的部分，这就是 ID

例如:
  完整 URL: https://cloud.dify.ai/app/xyz/workflows/my-workflow-uuid-here
                                                        ^^^^^^^^^^^^^^^^^^
  ID 是: my-workflow-uuid-here

复制这个 ID!

================================================================================
步骤 6: 对 Workflow2 重复 (步骤 2-5)
================================================================================

1. 回到应用列表
2. 找到 Workflow2 或 "生成答案"
3. 点击进入编辑
4. 看 URL 复制 workflows/ 后的 ID

================================================================================
步骤 7: 告诉我两个 ID
================================================================================

给我这样的信息:

  Workflow1 (生成问题) API ID: ________________________

  Workflow2 (生成答案) API ID:  ________________________

或者复制完整 URL:

  Workflow1 URL: https://cloud.dify.ai/app/...

  Workflow2 URL: https://cloud.dify.ai/app/...

================================================================================
如果找不到?
================================================================================

如果 URL 不是 cloud.dify.ai 格式，可能：

1. 你在公开分享页面 (udify.app)
   → 点击 "编辑" 按钮 (如果有的话)
   → 或在新标签打开 https://cloud.dify.ai

2. 工作流配置不完整
   → 尝试重新创建工作流

3. URL 格式完全不同
   → 截图发给我

================================================================================
验证你找到的是否正确
================================================================================

✓ 正确的 ID 特征:
  - 长度 > 10 字符
  - 由字母/数字组成 (可能有 - 或 _)
  - 看起来像: UUID 或产品 ID
  - 例子: 5a9c3d4b-7e9f-4f2a-8c1d-2e5f6a8b3c1d

✗ 错误的 ID:
  - 短于 8 字符
  - 看起来像: 560EB9DDSwOFc8As (这个是我们错误使用的!)
  - 只有字母数字没有连字符

================================================================================
立即开始!
================================================================================

1. 打开浏览器
2. 登录 https://cloud.dify.ai
3. 找到 Workflow1
4. 看 URL 的 workflows/ 部分
5. 复制并告诉我

大约需要 3 分钟! ⏱️
