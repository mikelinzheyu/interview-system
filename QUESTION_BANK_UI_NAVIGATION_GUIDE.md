## 🔐 权限检查清单

### 非登录用户

```
尝试访问 /learning-hub
→ 路由守卫检查: requiresAuth = true
→ 用户未登录
→ 自动重定向到 /login
```

**受保护的路由**（需要登录）：
- `/questions` → `/learning-hub`
- `/learning-hub`
- `/learning-hub/:domainSlug`
- `/learning-paths`
- `/learning-paths/:pathSlug`
- `/admin/questions/new` (也需要管理员)
- `/admin/questions/:id/edit` (也需要管理员)

### 普通用户（已登录，非管理员）

```
尝试访问 /admin/questions/new
→ 路由守卫检查: requiresAdmin = true
→ 用户不是管理员 (isAdmin = false)
→ 自动重定向到 /home
→ 控制台输出警告: "Access denied to /admin/questions/new: Admin privileges required"
```

**无法访问的路由**（需要管理员权限）：
- `/admin/questions/new`
- `/admin/questions/:id/edit`

### 管理员用户（已登录，是管理员）

```
✅ 可以访问所有路由，包括：
- /learning-hub
- /learning-hub/:domainSlug
- /learning-paths
- /learning-paths/:pathSlug
- /admin/questions/new
- /admin/questions/:id/edit
```

## 🎯 快速导航参考

### 用户场景1：学习题目
```
1. 进入首页 (/home)
2. 点击"学习中心"卡片
3. 选择推荐学科 或 从分类浏览中选择
4. 进入题库页面 (/learning-hub/:slug)
5. 浏览和学习题目
```

### 用户场景2：查看学习路径
```
1. 进入首页 (/home)
2. 点击"学习路径"卡片
3. 浏览各种学习路径
4. 点击路径卡片查看详情 (/learning-paths/:slug)
```

### 用户场景3：管理员创建题目
```
1. 进入首页 (/home)
2. 点击"题目管理"卡片 (仅管理员可见)
3. 填充表单创建新题目
4. 提交创建
```

### 用户场景4：管理员编辑题目
```
1. 从任何题库界面获取题目ID
2. 导航到 /admin/questions/{id}/edit
3. 修改题目内容
4. 保存修改
```

## 🔍 没有找到你要的功能？

### 常见问题

**Q: 我看不到"题目管理"卡片**
A: 这个功能仅对管理员可见。请确认你的账户具有管理员权限。

**Q: 我无法访问管理员路由**
A: 系统自动检查权限。如果你是管理员但仍无法访问，请：
1. 退出登录
2. 重新登录
3. 刷新页面

**Q: 从学习中心找不到返回按钮**
A: 在学习中心的"更多功能"下拉菜单中有"返回首页"选项。

**Q: 我想直接访问URL应该输入什么？**
A: 参考本指南的"路由"部分，但建议使用UI导航，这样更安全。
