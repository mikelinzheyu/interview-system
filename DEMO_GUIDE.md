# AI面试系统 - 页面演示指南

## 📺 演示功能说明

本指南将帮助你在 `http://localhost:5174/interview/ai` 页面上进行完整的AI面试系统演示。

---

## 🚀 快速开始演示

### 1. 访问AI面试页面
打开浏览器访问：http://localhost:5174/interview/ai

### 2. 打开浏览器开发者工具
- 按 `F12` 打开开发者工具
- 切换到 **Console** 标签页

### 3. 导入演示脚本
在控制台中运行以下代码来导入并执行演示：

```javascript
// 方法一：快速演示（5秒左右）
import { runQuickDemo } from '/src/utils/demoSimulator.js'
const app = document.querySelector('#app').__vue__
runQuickDemo(app)

// 方法二：完整演示（详细展示，约10秒）
import { runInterviewDemo } from '/src/utils/demoSimulator.js'
runInterviewDemo(app)
```

## 🔧 高级用法

### 自定义演示参数

你可以修改 `demoSimulator.js` 中的参数来自定义演示：

```javascript
// 修改问题内容
componentInstance.currentQuestion.question = '你的自定义问题'

// 修改用户回答
componentInstance.finalTranscript = '你的自定义回答'

// 修改分析评分
componentInstance.analysisResult.overallScore = 90

// 修改评分颜色（根据分数自动调整）
getScoreColor(90) // 返回绿色
getScoreColor(75) // 返回橙色
getScoreColor(60) // 返回红色
```

### 重置演示

```javascript
import { resetDemo } from '/src/utils/demoSimulator.js'
resetDemo(app)
```

## 🐛 常见问题

### Q: 演示脚本无法导入？
**A:** 确保你在 `http://localhost:5174/interview/ai` 页面上，并且前端服务正在运行。

### Q: 页面没有更新演示数据？
**A:** 尝试手动触发Vue的响应式更新：
```javascript
app.$forceUpdate()
```

### Q: 想要修改演示的题目和回答？
**A:** 编辑 `demoSimulator.js` 中的 `currentQuestion` 和 `finalTranscript` 变量。

### Q: 演示数据消失了？
**A:** 刷新页面会重置所有数据。你可以再次运行演示脚本。
